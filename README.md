# Replication-of-The-Miracle-of-Microfinance-by-Banerjee-et-al.-2025-
This was a replication of Banarjee's paper with regression visualization and analysis. Included is the code we used to emulate the OLS regressions, the original datasets, and a comparison to the original regression visualization. 

Also attached to the files is the write-up we included for an in-depth economic review of the paper from our reproduction.
